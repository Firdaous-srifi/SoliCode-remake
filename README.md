SoliCode Website Remake. This is a project where I have remade the SoliCode website for my school using HTML CSS and JavaScript. The goal of this project was to replicate the functionality and design of the original website while adding enhancements to improve the user experience. Features: Responsive Design: The website is fully responsive and adapts to different screen sizes making it user-friendly on mobile tablet and desktop devices. Clean UI: The design has been reimagined with a focus on simplicity and ease of navigation. Interactive Elements: JavaScript is used to create interactive features such as buttons forms and navigation. Well-structured HTML/CSS: Organized and semantic HTML structure combined with modern CSS techniques for styling. Technologies Used: HTML: For creating the structure of the website. CSS: To style the website and make it visually appealing. JavaScript: To implement dynamic features and interactivity. Purpose: This project is meant to provide a modern version of the SoliCode website tailored for my school community. It aims to showcase my web development skills and contribute to the school's digital presence. Feel free to explore and contribute to this repository. Any suggestions or improvements are welcome!
